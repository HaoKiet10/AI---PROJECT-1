* ĐỂ CHẠY THUẬT TOÁN:
    - Bước 1: vào hàm main.py của thư mục Project-01/Sources, tại đây có biến tên inputFileName. 
Bạn hãy sửa tên ứng với tên file input bạn muốn chạy. 
        VD: Bạn muốn chạy testcase số 2 thì sửa thành: inputFileName = "../Input/Input-2.txt"
    - Bước 2: Các bạn mở terminal lên và cd vào thư mục Project-01/Sources. Sau đó gõ python main.py or python3 main.py
    - Bước 3: Mở thư mục Output để xem kết quả của các file output.txt tướng ứng.


* ĐỂ CHẠY GIAO DIỆN:
    - Bước 1: Các bạn tải thư viện pygame về bằng pip install pygame or pip3 install python
    - Bước 2: Các bạn mở terminal lên và cd vào thư mục Project-01/UI/Sources. Sau đó gõ python Home.py or python3 Home.py
    - Bước 3: Cửa sổ của giao diện sẽ được hiện lên và trãi nghiệm game.
